Clases usadas para la Barra de Navegación

navbar: Define la barra de navegación. navbar-expand-lg:controla cuándo el contenido de la barra se colapsa y cuándo se expande automáticamente. Antes del tamaño lg (large): la barra de navegación estará colapsada, es decir, aparecerá el ícono de "hamburguesa" para abrir el menú. A partir del tamaño lg en adelante: la barra de navegación se expande y muestra todos los enlaces sin colapsar, ocupando su lugar completo en la pantalla. navbar-dark: le aplica estilos oscuros, haciendo que el texto y los íconos sean más visibles. bg-#6C757D: Color de fondo personalizado.

Todos los links y el buscador están dentro de un div con la clase container-fluid:asegura que los elementos dentro de la barra de navegación se distribuyan uniformemente a lo largo del viewport, adaptándose a pantallas más pequeñas o grandes sin dejar márgenes laterales.

Botón de menú en tamaño MD y SM
navbar-toggler: da clase al botón que se usa para abrir/cerrar el menú que aparece en pantallas pequeñas cuando la barra de navegación está colapsada. data-bs-toggle="collapse": es un atributo de Bootstrap que indica que este botón controlará la funcionalidad de "colapsar" (expandir o contraer) un

menú o un panel. data-bs-target="#navbarNavAltMarkup": este atributo indica qué elemento será colapsado o expandido. En este caso, el div con el id="navbarNavAltMarkup" (que contiene los enlaces de navegación). Esto hace que al hacer clic en el botón, el menú se despliegue o se oculte. aria-controls="navbarNavAltMarkup": atributo de accesibilidad que indica que el botón controla el menú con el ID navbarNavAltMarkup. aria-expanded="false" : indica que el menú está colapsado inicialmente (no expandido). Este valor cambia dinámicamente a true cuando el menú está desplegado. aria-label="Toggle navigation": es un atributo crucial para la accesibilidad.

Ícono de menú

span class="navbar-toggler-icon" : es un elemento visual que representa el ícono de hamburguesa (las tres líneas paralelas) que se utiliza comúnmente para indicar que hay un menú colapsado.
Los links que en pantallas MD y SM forman parte del menú, asi como el buscador y el link de "Opciones", se encuentran dentro de un div con clase collapse navbar-collapse: Colapsa el contenido de la barra de navegación en tamaños pequeños y lo expande en tamaños LG. navbar-nav me-auto: Aplica margen a la derecha de los enlaces en la barra de navegación (esto permite que el formulario de búsqueda esté a la derecha en pantallas grandes).

Campo de busqueda

d-flex: Utilizado para alinear correctamente el campo de búsqueda dentro de la barra de navegación. ms-lg-auto: Aplica un margen a la izquierda automático solo en pantallas LG para empujar el formulario hacia la derecha. order-md-first order-lg-last: Cambia el orden del campo de búsqueda: En pantallas MD y SM, se posiciona al inicio del menú (primero). En pantallas LG, se coloca al final (último) para mantenerse a la derecha. form-control: Estiliza el campo de búsqueda, haciéndolo visualmente acorde con el resto de la barra de navegación. me-2: Aplica margen a la derecha del campo de búsqueda para darle espacio con otros elementos.

Enlaces del menú

nav-link: estiliza los enlaces de navegación. mx-5: Aplica márgenes horizontales entre los enlaces. d-md-block d-lg-none: El enlace "Opciones" es visible solo en pantallas MD y SM. Se oculta en pantallas LG. disabled: Desactiva el enlace "Disabled", lo hace no clickeable y aplica un estilo visual de desactivado.

Título y Textos Aleatorios

class="carousel slide" data-bs-ride="carousel" data-bs-interval="2000": Define el carrusel con cambios cada 2 segundos. d-none d-lg-block: Oculta el título "Bootstrap" y el carrusel de textos aleatorios en pantallas SM y MD. Solo es visible en pantallas LG.

Sistema de Grid

container : Contenedor principal que encapsula los párrafos y el acordeón. row: Clase que define una fila en el sistema de cuadrícula de Bootstrap. col-lg-9 : Los párrafos ocupan 9 columnas en pantallas grandes (LG), dejando 3 columnas para el acordeón.

Acordeón de Opciones

accordion: Define el contenedor como un acordeón, que puede expandirse y colapsarse. collapse: Inicialmente colapsa el acordeón. col-lg-3: El acordeón ocupa 3 columnas del sistema de cuadrícula en pantallas grandes (LG). d-lg-block: Muestra el acordeón solo en pantallas grandes (LG). rounded: Aplica bordes redondeados a cada opción del acordeón. p-3: Aplica padding (espaciado interno) a cada opción del acordeón.

Párrafos
col-12 col-md-6 col-lg-4 : Cada párrafo ocupa: 12 columnas en pantallas pequeñas (SM), tomando todo el ancho. 6 columnas en pantallas medianas (MD), mostrando dos columnas de párrafos. 4 columnas en pantallas grandes (LG), mostrando tres columnas de párrafos. fondo : Clase personalizada que puedes usar para aplicar un fondo a los párrafos. mb-3 : Aplica un margen inferior a cada párrafo para crear espacio entre ellos.
